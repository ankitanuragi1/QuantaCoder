## 🚀 Getting Started

### 1️⃣ Clone the repository
```bash
git clone https://github.com/ankitanuragi1/QuantaCoder.git
````
### 2️⃣ Install dependencies

```bash
npm install
```
### 3️⃣ Run development server

```bash
npm run dev
```
### 4️⃣ Build for production
```bash
npm run build
```
